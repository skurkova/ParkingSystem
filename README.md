# 🅿️ Система управления парковкой

Система автоматического управления парковкой с распознаванием номера и автоматической оплатой.

**Приложение позволяет:**
- Регистрировать клиентов с привязанной картой и автомобилем
- Создавать парковочные зоны с указанием количества мест
- Отслеживать заезд/выезд автомобилей на парковку
- Автоматически проверять наличие свободных мест
- Контролировать привязку платёжной карты при выезде

## 🛠️ Технологический стек

- **Язык программирования**: Python 3.9 
- **Веб-фреймворк**: Flask 
- **ORM**: Flask-SQLAlchemy 
- **База данных**: SQLite 
- **Тестирование**: pytest 
- **Генерация данных**: Factory Boy, Faker 
- **CI/CD**: GitLab CI/CD
- **Проверка качества**: mypy, black, isort, flake8

## 🚀 Установка и запуск
#### 1. Клонирование репозитория
`git clone git@github.com:skurkova/ParkingSystem.git`

`cd ParkingSystem`
#### 2. Создание и активация виртуального окружения
`python3 -m venv venv`

`source venv/bin/activate`
#### 3. Установка зависимостей
`pip install -r requirements.txt`
#### 4. Запуск приложения
` python3 -m scr.app`
#### 5. Приложение будет доступно по адресу: 
http://localhost:8080

## 📡 Эндпоинты приложения
### Основные сущности
Приложение работает с тремя моделями:
- `Client` — клиент с данными и платёжной картой
- `Parking` — парковочная зона с местами
- `ClientParking` — заезд/выезд

### Приложение предоставляет следующие API-эндпоинты:
`GET` `/`: Заполнение БД тестовыми данными

`GET` `/clients`: Список всех клиентов

`GET` `/clients/<client_id>`: Информация о клиенте по ID

`POST` `/clients`: Создание нового клиента

`POST` `/parkings/`: Создание новой парковки

`POST` `/client_parkings`: Заезд на парковку

`DELETE` `/client_parkings`: Выезд с парковки

## 🧪 Запуск тестов
### Запуск всех тестов
`pytest tests/`
### Запуск тестов с покрытием кода
`pytest tests/ --cov=scr`

## 🔁 CI/CD пайплайн
Проект изначально разрабатывался с использованием `GitLab CI/CD` для автоматической проверки качества кода при каждом создании Merge Request. 
Конфигурация сохранена в файле `.gitlab-ci.yml` и включает:
- Запуск тестов через `pytest`
- Статическую типизацию через `mypy`
- Форматирование кода через `black` и `isort`
- Линтинг через `flake8`
